한도값 보정 방법

현재 카드의 한도는 기본 추정값입니다. 본인 플랜의 실제 한도에 맞추려면 Claude Code /usage 퍼센트로 1회 보정하세요.

  1. Claude Code에서 /usage 실행 → Session(5hr) %, Weekly(7day) %, 플랜 확인 (확인한 시각도 기억)
  2. 아래 템플릿을 복사해 빈칸(%·시각·플랜)을 채운 뒤, Claude Code 채팅에 붙여넣기
  3. Claude가 그 시각까지의 weighted 합계를 역산해 config.json의 FIVE_HOUR_LIMIT/WEEKLY_LIMIT과 calibrated/calibratedAt을 갱신 → 브라우저 새로고침
보정 프롬프트 템플릿 
한도값 보정할게. 방금 [HH:MM]에 /usage 확인한 결과:
- Session (5hr): __%
- Weekly (7 day): __%
- 플랜: ____

[HH:MM] 시점까지 기록된 토큰만으로 5시간 윈도우 weighted 합계와
주간 weighted 합계를 계산해줘. 그 이후 기록은 합산에서 제외.
위 퍼센트와 일치하도록 FIVE_HOUR_LIMIT과 WEEKLY_LIMIT을 역산해서
config.json에 반영하고, calibrated=true / calibratedAt도 갱신해줘.
계산 근거(기준 시각, 토큰 합계, 산식)도 보여줘.
