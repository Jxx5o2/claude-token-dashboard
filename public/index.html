한도값 보정 방법

현재 카드의 한도는 기본 추정값입니다. 복잡한 계산 없이 Claude Code에게 맡기면 됩니다. 필요한 건 /usage 화면과 확인한 시각뿐이에요.

  1. Claude Code에서 /usage 실행
  2. 그 화면을 스크린샷 찍어 Claude Code 채팅에 붙여넣기 (또는 Session/Weekly 퍼센트만 타이핑)
  3. 아래 한 줄로 요청 → Claude가 config.json 한도값·보정 상태를 알아서 갱신 → 브라우저 새로고침
이렇게 요청하세요 (복사해서 붙여넣기) 
이 /usage 기준으로 한도값 보정해줘.
확인 시각은 [OO시 OO분]이야.
config.json의 FIVE_HOUR_LIMIT / WEEKLY_LIMIT 과
calibrated / calibratedAt 을 갱신하고, 근거도 알려줘.
